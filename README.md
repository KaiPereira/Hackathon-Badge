# Hackathon Badges

These are alchemical, cosmological, PCB hackathon badges I designed in a day for the Fallout hackathon in Shenzhen. Powered by the CH32 MCU, they stack ontop of each other using testpoints snd shine through each other to create a dun effect and have cool designs on them!
